# Reference knowledge base (docs/ref)

Put manuals, notes, papers, and links here. The goal is to make **all math/modeling choices traceable to sources**.

## How to add new sources

For each source, add a short entry:

- **Title**
- **Version / date**
- **Where**: file path in `docs/ref/...` and/or external link
- **What we use it for**: element models, Lie conventions, coordinate definitions, etc.
